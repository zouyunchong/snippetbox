# Privacy Policy SnippetBox does not collect, store, or transmit personal data to external servers. All saved snippets are stored locally in the user's browser using IndexedDB. The extension does not: * Track browsing history * Sell user data * Upload saved content to cloud servers * Share data with third parties Permissions are only used to: * Access selected text on webpages * Save snippets locally * Display extension functionality If future cloud synchronization or AI features are introduced, this privacy policy will be updated accordingly.
